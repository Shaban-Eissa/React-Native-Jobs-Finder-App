## Installation

1. Clone the repository

   ```bash
   git clone https://github.com/Shaban-Eissa/React-Native-Jobs-Finder-App
   ```

2. Navigate to the project directory
    
    ```bash
    cd React-Native-Jobs-Finder-App
    ```
   
3. Change to folder for the app you want, Install dependencies
    
    ```bash
    npm install
    ```
4. Create `.env` file : open [Rapid Api](https://rapidapi.com/) then click on `search api's` and write JSearch and after choose it you will see left panel contain endpoints for the api and right panel contain the api key's. You will choose `job search` endpoint from left sidepanel then choose code snippets from right panel (Target 'Node.js' , Client 'Axios')
   ```bash
   EXPO_PUBLIC_RAPID_API_KEY= x-rapidapi-key
   EXPO_PUBLIC_RAPIDAPI_HOST= x-rapidapi-host
   ```

## Usage

1. Start the UI:
    ```bash
    npx expo start
    ```
